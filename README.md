# 🧾 財審 AI · 派款前財務核實系統

創業者（客戶）喺後台提交公司資料同報表（營業執照、銷售收入、流水帳、財務報表），
AI Agent 會勾稽佢嘅**收入來源同財務報表係咪真實**，計出一個核實分數，
再根據分數建議**係咪批准派發款項**。

## 三步流程

1. **提交資料** — 填公司基本資料、上傳報表（有位俾你交文件）、輸入財務數據。
2. **AI 分析** — AI Agent 逐項核實，過程透明可見。
3. **核實報告** — 出核實分數（0–100）、批准/覆核/拒絕決策、建議派發金額，同每項核實嘅解釋。

## 上傳報表位

四個上傳位，支援 PDF / 圖片 / Excel（可多份）：

- 🪪 營業執照
- 📊 財務報表（損益表・資產負債表・現金流量表）
- 🏦 銀行流水帳
- 🧾 銷售合同 / 發票

> 示範版本純前端運行，文件只喺你部機讀取，**唔會上傳到任何伺服器**。

## AI 核實邏輯（100 分）

| 核實項目 | 分數 | 睇乜嘢 |
|---|---|---|
| 文件完整性 | 20 | 四類必要文件係咪齊 |
| 三表勾稽一致性 | 25 | 資產=負債+權益；淨利潤 vs 經營現金流 |
| 收入真實性 | 25 | 收入 vs 銀行流水；應收帳款增速 vs 收入增速；毛利率合理性 |
| 收入來源健康度 | 15 | 最大客戶集中度 |
| 稅務合理性 | 15 | 綜合稅負率是否與規模相稱 |

另加 **Benford 首位數字鑑證** 作輔助信號，偵測數字係咪「砌」出嚟。

**決策門檻**：≥80 批准派發（100%）｜65–79 有條件批准/需人工覆核（50%）｜<65 暫不派款（0%）。

> 想快速睇效果：撳「載入示範數據」（健康公司）或「載入可疑樣本」（疑似造假）即可。

## 技術

- 純前端單一 HTML 檔，零依賴、零後端、免費。
- 分析引擎為規則化財務鑑證邏輯，**可解釋、非黑盒**；日後可接大模型做文件 OCR 與敘述式分析。
